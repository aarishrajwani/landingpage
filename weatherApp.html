Personal Weather Project

After learning about how API's work, what they are used for, and how powerful they are, I wanted to try and create my own weather app that calls from an API. For this project, I created an account wth OpenWeatherMap to get an api. I then sketched and coded the UI in html and css. I then called the weather api using Java Script to take a user's searches and run them through the api. In addition to working with the weather api, I also included background images on the weather app that calls Unsplash for an image based on the city the user typed in. The app is also fully responsive to different screen sizes.
